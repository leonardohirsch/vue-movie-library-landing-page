# Amadeus movie-library

App in production:
[https://leopruebas.com.ar/amadeus/](https://leopruebas.com.ar/amadeus/)

## Stack
-Vite
-Vue 3
-Pinia
-TailwindCSS
-Heroicons
-Vue-Select

## Instalation

### Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
